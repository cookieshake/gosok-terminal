## 1. Ctrl+A / Ctrl+F 터미널 포워딩 (데스크톱 브라우저)

**관련 커밋:** `538d5a4`, `5981aed`

**구현 내용:**
- 데스크톱 브라우저에서 `Ctrl+A`, `Ctrl+F`가 브라우저 기본 동작(전체선택, 페이지검색)으로 가로채지는 문제 수정
- macOS는 `Cmd+V` / `Cmd+F`, 그 외는 `Ctrl+V` / `Ctrl+F`로 라우팅
- 플랫폼 감지(`navigator.platform`, `navigator.userAgent`)로 Mac 여부 판별

**spec 필요 내용:** 키보드 이벤트 라우팅 정책 — 어떤 키가 브라우저로, 어떤 키가 터미널로 전달되는지.

**추가 위치:** `spec/features/terminal.md` [TERM.5], `spec/scenarios/terminal.md` [SC.TERM.5]

## 2. WebSocket 재접속 후 실시간 알림 복원

**관련 커밋:** `6af2210`

**spec 필요 내용:** notification-center-design.md에 WebSocket 재접속 시나리오 및 구독 복원 동작 추가.

**추가 위치:** `spec/features/websocket.md` [WS.4], `spec/scenarios/websocket.md` [SC.WS.7]

## 3. 모바일 키보드 닫힘 시 스크롤 위치 초기화

**관련 커밋:** `a2c8f53`

**구현 내용:**
- iOS/Android에서 소프트 키보드가 닫힐 때 `window.scrollY`가 비정상 위치에 머무는 버그 수정
- `visualViewport` resize 이벤트 감지 후 `window.scrollTo(0, 0)` 호출

**spec 필요 내용:** 모바일 UX 동작 명세 (현재 spec 없음).

**추가 위치:** `spec/features/terminal.md` [TERM.6], `spec/scenarios/terminal.md` [SC.TERM.6]
